最优二叉树：

		给定n个权值作为n的叶子节点，构造一棵二叉树，若带权路径长度达到最小，称这样的二叉树为最优二叉
		树，也称哈夫曼树


		例：给定4个叶子结点a，b，c和d，分别带权7，5，2和4。构造如下图所示的三棵二叉树(还有许多棵)，
			它们的带权路径长度分别为：


			a：
				【】
			   ——————
			  |      |
			【】     【】
		  ———————   ——————
		 7|    5|  2|    4|
		  a     b   c     d



			b：
				【】
			   ——————
			  |      |2
			【】      c
		  ———————
		 4|     | 
		  d    【】
			  ——————
			 7|    |5
			  a    b 


			c：
				【】
			   ——————
			 7|      |
			  a     【】
					—————
				   5|   |
					b  【】
					   ————
					  2|   |4
					   c   d



		(a) WPL= 7*2 + 5*2 + 2*2 + 4*2 = 36
        (b) WPL= 7*3 + 5*3 + 2*1 + 4*2 = 46
        (c) WPL= 7*1 + 5*2 + 2*3 + 4*3 = 35

		其中(c)树的WPL最小，可以验证，它就是哈夫曼树
